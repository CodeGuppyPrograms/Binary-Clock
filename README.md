Learn how to implement a software simulation of a binary BCD clock. This is a very good exercise to learn about the binary numeration system.

![Image](thumb.png)

Feel free to customize the clock and then include it in your web site or blog.
 
## Source code 
The JavaScript source code of this program can be found [here](sketches/program.js). To run it, you need to import it in the [codeguppy.com](https://codeguppy.com) online editor. Alternatively, you can use the source code in your p5.js sketches (advanced use). 
## Online version 
To see the code running, check the online playground at [https://codeguppy.com/code.html?t=binary_clock](https://codeguppy.com/code.html?t=binary_clock)
